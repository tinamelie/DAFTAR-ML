================================================================================
IMPORTANT: EXPLANATION OF FEATURE RANKING METHODS
================================================================================

DAFTAR-ML uses multiple methods to rank feature importance:

1. SAMPLE-LEVEL IMPACT (shown in shap_beeswarm_sample.png)
   * Calculated directly from raw SHAP values across all samples
   * Shows features with strongest overall impact
   * Can highlight features important in specific contexts

2. FOLD-LEVEL IMPACT (shown in shap_beeswarm_fold.png)
   * Calculated by averaging SHAP values within each fold first
   * Shows features that are consistently important across data splits
   * More robust to random variation in the data

================================================================================
INTERPRETATION GUIDE
================================================================================

SHAP values represent the impact of each feature on model predictions.
- Positive values mean the feature pushes predictions higher
- Negative values mean the feature pushes predictions lower
- The magnitude indicates how strong the effect is
- Standard deviation (±) shows the variability of this effect

These values are calculated based on SHAP (SHapley Additive exPlanations),
which analyzes each feature's contribution to predictions across the dataset.
